# skillAuth
项目结构:</br>
本项目后台采用Spring 4.0 + Mybatis3 + Spring MVC</br>
数据库:MySQL</br>
JAVA版本:1.8</br>
应用服务器:Tomcat8.0</br>
数据库服务器:MySqlServer5.6</br>
前端框架:BootStrap</br>

项目介绍:</br>
鉴定信息处理系统是针对技能技能鉴定信息管理而提出的能够满足学校业务流程的需求。
本项目的需求中，包括了考证管理和实训室管理两大模块。具有业务流程较为复杂、应用前
景广的特点。实现了管理信息化，提高了管理效率，避免了人工操作而造成的差错。
